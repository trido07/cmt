### Environment variables

- PORT: app port
- DB_USER: postgres db username
- DB_PASSWORD: postgres db password
- DB_DB: postgres db
- DB_HOST: postgres db
- DB_PORT: postgres db
- CORS_ORIGIN: cors origin
- NODE_ENV: production or development
- PG_LOCAL_DATA_FOLDER (development purpose only): connect psql data folder to local folder if run psql on docker
- JWT_SECRET: jasonwebtoken secret
- JWT_EXPIRATION: jasonwebtoken expiration
- MASTER_USERNAME: master account username
- MASTER_PASSWORD: master account password
